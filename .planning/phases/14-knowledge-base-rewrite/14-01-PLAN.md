Rewrite knowledge_base/general.md to contain the full AGM negotiation playbook. Keep the existing structure sections that work (Core Principles, Tone by Stage, Do NOT Say, Deliverable Terminology) but expand significantly with AGM-specific strategy content.

Add these new sections:

**## Campaign Goal Anchoring** - Document the three campaign goal types (organic performance, paid performance, balance) and how to anchor negotiation approach based on goal.

**## Negotiation Levers** - Document the 4 main levers with detailed guidance:
1. Deliverable Tiers: Trade down from Scenario 1 (most deliverables) to Scenario 2 to Scenario 3 (minimum). Explain how reducing deliverable count/type reduces cost while preserving value.
2. Usage Rights: Trade duration down from target to minimum (e.g., 12 months to 6 months to 3 months). Shorter duration = lower cost.
3. Product/Upgrade: Offer product value, monetary value of product, as added compensation when cash rate is at ceiling. Include upgrade/premium versions as sweetener.
4. CPM Sharing: Selectively share CPM target with motivated influencers to justify rate constraints. Only use with influencers who seem data-driven and engaged.

**## Budget Maximization Strategy** - Document: open high (request more deliverables at lower per-unit rate), create concession room, use CPM target to calculate and justify boundaries. Include the concept of per-influencer cost floor (never go below) and max-without-approval ceiling (escalate above).

**## Content Syndication** - Explain cross-posting as added value (e.g., IG Reel cross-posted to TikTok counts as both platform deliverables without requiring unique content per platform).

**## Graceful Exit Protocol** - How to politely decline when economics don't work: preserve relationship, leave door open for future, never burn bridges. Be specific about language and framing.

**## Direct Influencer vs Talent Manager** - Brief guidance note: talent managers expect data-backed, transactional style; direct influencers respond to relationship-driven, creative-alignment language. (Full counterparty intelligence is Phase 16, but basic awareness belongs in the playbook.)

Update the existing "Tone by Negotiation Stage" section to align with NegotiationState enum values used in code: initial_offer, counter_received, counter_sent, agreed, escalated. Add tone guidance for each state.

Keep the existing Do NOT Say list and Deliverable Terminology sections, expanding if needed.

**YAML frontmatter schema for every example file:**
```yaml
---
scenario: string        # unique scenario ID (e.g., "positive_close", "bundled_rate")
title: string           # human-readable title (e.g., "Positive Close - Agreement Confirmation")
stages:                 # list of NegotiationState values where this example is relevant
  - counter_sent        # maps to NegotiationState enum values
  - agreed
tactics:                # list of tactic keywords used in this email
  - acknowledgment
  - next_steps
platform: string|null   # "instagram", "tiktok", "youtube", or null for platform-agnostic
---
```

**The 9 examples to create (based on AGM strategy docs provided in planning context):**

1. **positive_close.md** - Agreement confirmation email. Stages: [agreed]. Tactics: [confirmation, next_steps, payment_terms]. Shows professional close with clear SOW recap and next steps.

2. **escalation.md** - Rate exceeds approval ceiling, escalating to human. Stages: [counter_received, escalated]. Tactics: [empathy, pause, escalation_signal]. Email acknowledges influencer rate and signals need to check with team/management before responding.

3. **walk_away.md** - Polite decline when budget gap is too large. Stages: [counter_received]. Tactics: [gratitude, relationship_preservation, future_door]. Graceful exit preserving relationship.

4. **bundled_rate.md** - Counter-offer with bundled multi-deliverable rate. Stages: [initial_offer, counter_sent]. Tactics: [bundling, value_framing, deliverable_tiers]. Shows IG Reel + TikTok bundle with per-deliverable breakdown and total rate.

5. **cpm_mention.md** - Using CPM data to justify rate constraints. Stages: [counter_sent, counter_received]. Tactics: [data_backed, cpm_justification, transparency]. Shows sharing CPM target with data-driven influencer.

6. **misalignment_exit.md** - Exit when deal economics fundamentally don't work. Stages: [counter_received]. Tactics: [honesty, no_pressure, relationship_preservation]. Different from walk_away: this is when there's a fundamental misalignment (e.g., influencer wants exclusivity we can't offer, or rate is 3x+ over ceiling).

7. **product_offer.md** - Offering product/upgrade as additional value. Stages: [counter_sent]. Tactics: [product_leverage, added_value, creative_framing]. Shows offering product when cash rate is maxed.

8. **usage_rights.md** - Negotiating usage rights duration downward. Stages: [counter_sent, counter_received]. Tactics: [usage_rights_trading, duration_reduction, cost_savings]. Shows trading from 12-month to 6-month usage rights to reduce rate.

9. **multi_platform_bundle.md** - Multi-platform bundle with content syndication. Stages: [initial_offer, counter_sent]. Tactics: [syndication, cross_posting, multi_platform, bundling]. Shows IG + TikTok bundle where IG Reel is cross-posted to TikTok.

**For each email example, include:**
- YAML frontmatter (as above)
- `## Context` section: 2-3 sentences describing when to use this example
- `## Email` section: The actual email body (use {placeholder} for dynamic values like {name}, {rate}, {deliverables})
- `## Key Tactics` section: Bulleted list explaining what tactics are demonstrated and why

**Important:** Use realistic AGM-style language throughout. Professional but warm. Partnership-first. Acknowledge creator value. Concise (3-5 paragraphs per email). Reference the strategy from the rewritten general.md playbook.

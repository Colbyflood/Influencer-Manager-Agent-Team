       - Determine the frontend dist path: `Path(__file__).resolve().parent.parent.parent / "frontend" / "dist"`
       - Only mount if the dist directory exists (graceful no-op in dev mode when using Vite dev server)
       - If dist exists:
         a. Mount StaticFiles at `/dashboard/assets` pointing to `dist/assets` (for JS/CSS bundles)

    IMPORTANT: The dashboard mount MUST come after all API route registrations to avoid the catch-all intercepting API requests. FastAPI matches routes in order of registration.

    IMPORTANT: Do not use `app.mount("/dashboard", StaticFiles(..., html=True))` alone because it won't handle SPA client-side routing (sub-paths like /dashboard/campaigns/123 would 404). Instead use the explicit catch-all route pattern that always returns index.html.

**1. Add `derive_cpm_bounds` function to `engine.py`:**

```python
def derive_cpm_bounds(
    cpm_target: Decimal | None = None,
    cpm_leniency_pct: Decimal | None = None,
) -> tuple[Decimal, Decimal]:
    """Derive CPM floor and ceiling from campaign target and leniency.

    If cpm_target is provided:
      floor = cpm_target (we open at target)
      ceiling = cpm_target * (1 + leniency_pct/100)
    If cpm_target is None: fall back to CPM_FLOOR/CPM_CEILING defaults.

    Returns: (cpm_floor, cpm_ceiling)
    """
```

The leniency percentage defines how far ABOVE the target we can go. A CPM Target of $25 with 20% leniency means floor=$25, ceiling=$30.

If `cpm_leniency_pct` is None but `cpm_target` is provided, default leniency to 0% (ceiling = target, meaning no room above target).

Keep `CPM_FLOOR` and `CPM_CEILING` constants as module-level defaults (they are imported elsewhere). They remain the fallback when no campaign data exists.

  <action>
1. Add a new "Actions" column header to the table (after the Agency column).

2. Create helper constants at the top of the component (inside the function):
   ```
   const TERMINAL_STATES = new Set(["agreed", "rejected", "stopped"]);
   const PAUSABLE_STATES = new Set(["awaiting_reply", "counter_sent", "counter_received", "initial_offer", "escalated", "stale"]);
   ```

3. Add state for tracking in-flight control actions:
   ```
   const [actionInFlight, setActionInFlight] = useState<string | null>(null);
   ```

4. Create a `handleControl` async function that:
   - Takes `threadId: string`, `action: "pause" | "resume" | "stop"`, and `event: React.MouseEvent` as params
   - Calls `event.stopPropagation()` to prevent row click navigation
   - Sets `actionInFlight` to `threadId`
   - POSTs to `/api/v1/campaigns/${campaignId}/negotiations/${threadId}/${action}`
   - On success, re-fetches the campaign detail data (call the same fetch logic from the useEffect, extract it into a `fetchDetail` function accessible from both useEffect and handleControl)
   - On error, show an `alert()` with the error message
   - Always clears `actionInFlight` in finally block

5. Refactor: Extract the fetch logic from the useEffect into a standalone `fetchDetail` async function defined with `useCallback` so it can be called from both useEffect and handleControl. The useEffect should call `fetchDetail()` and the cleanup should set `cancelled = true`. Use a ref for the cancelled flag.

6. In each table row, add an Actions `<td>` cell that renders buttons based on state:
   - If `neg.state` is in PAUSABLE_STATES: show "Pause" button (yellow/amber) and "Stop" button (red)
   - If `neg.state === "paused"`: show "Resume" button (green) and "Stop" button (red)
   - If `neg.state` is in TERMINAL_STATES: show a dash "--" or nothing (no actions available)

   Button styling (Tailwind):
   - Pause: `px-2 py-1 text-xs font-medium rounded bg-amber-100 text-amber-800 hover:bg-amber-200 disabled:opacity-50`
   - Resume: `px-2 py-1 text-xs font-medium rounded bg-green-100 text-green-800 hover:bg-green-200 disabled:opacity-50`
   - Stop: `px-2 py-1 text-xs font-medium rounded bg-red-100 text-red-800 hover:bg-red-200 disabled:opacity-50`

   All buttons should be `disabled` when `actionInFlight === neg.thread_id`.

7. Each button's onClick should call `handleControl(neg.thread_id, "pause"|"resume"|"stop", event)`.

Create `src/negotiation/llm/sow_formatter.py` with two public functions:

1. `format_rate_adjustment(original_rate: str, adjusted_rate: str) -> str`
   - Returns strikethrough formatted rate string: `~~$2,000~~ $1,500`
   - If original_rate == adjusted_rate, just returns `$1,500` (no strikethrough)
   - Formats rates with comma thousands separators and .00 cents

2. `format_sow_block(deliverables_summary: str, usage_rights_summary: str | None, rate_display: str, platform: str) -> str`
   - Builds a structured SOW text block matching AGM email format:
     ```
     Scope of Work:
     - [deliverable items, one per line from deliverables_summary]
     - Usage Rights: [usage_rights_summary or "per standard terms"]
     - Rate: [rate_display]
     ```
   - Parses deliverables_summary string (comma-separated or multi-line) into bullet points
   - Uses platform-correct terminology (from knowledge_base general.md Deliverable Terminology section)

Update `src/negotiation/llm/prompts.py`:

1. Update `EMAIL_COMPOSITION_SYSTEM_PROMPT` to enforce AGM style:
   - Add explicit style rules: "Write in AGM partnership style -- warm but professional. Open by acknowledging the creator's value or referencing their content. Keep to 3-5 concise paragraphs. Use first-person plural ('we') to represent the team."
   - Add SOW formatting instruction: "When a FORMATTED SOW BLOCK is provided, embed it in the email as-is. Do not reformulate the SOW structure or rate values -- use them exactly as given."
   - Keep existing rules about exact rates, no promises, no other influencer references

2. Update `EMAIL_COMPOSITION_USER_PROMPT` to include a `{sow_block}` placeholder after the deliverables section:
   ```
   FORMATTED SOW BLOCK (embed this in your email as-is):
   {sow_block}
   ```

- `test_format_sow_block_no_usage_rights` -- defaults to "per standard terms"

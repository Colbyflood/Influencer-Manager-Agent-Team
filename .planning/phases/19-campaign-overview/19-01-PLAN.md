<task type="auto">
  <name>Task 1: Create campaign list API endpoint with status aggregation</name>
  <files>src/negotiation/api/__init__.py, src/negotiation/api/campaigns.py</files>
  <action>
Create `src/negotiation/api/__init__.py` as empty package init.

Create `src/negotiation/api/campaigns.py` with an APIRouter:

1. Define Pydantic response models:
   - `CampaignMetrics`: avg_cpm_achieved (float | None), pct_closed (float), budget_utilization (float | None)
   - `CampaignStatusCounts`: active_negotiations (int), agreed (int), escalated (int), rejected (int), total_influencers (int)
   - `CampaignSummary`: campaign_id (str), client_name (str), platform (str), budget (float), status_counts (CampaignStatusCounts), metrics (CampaignMetrics)
   - `CampaignListResponse`: campaigns (list[CampaignSummary]), total (int)

2. Create `GET /campaigns` endpoint on the router:
   - Access `negotiation_states` from `request.app.state.negotiation_states`
   - Group negotiations by `campaign_id` from each entry's `context["campaign_id"]`
   - For each campaign group:
     a. Count states: active = states in (INITIAL_OFFER, AWAITING_REPLY, COUNTER_RECEIVED, COUNTER_SENT), agreed = AGREED, escalated = ESCALATED, rejected = REJECTED. total_influencers = len of group.
     b. Compute pct_closed = (agreed + rejected) / total_influencers * 100
     c. Extract avg_cpm from the CampaignCPMTracker's `running_average_cpm` property (convert Decimal to float). Use the cpm_tracker from the first entry in the group (they share the same tracker instance per campaign).
     d. Budget utilization: sum of agreed CPM amounts from tracker._agreements vs campaign.budget. If campaign.budget > 0: (sum_of_agreed / budget) * 100, else None.
     e. Get campaign_id, client_name, platform, budget from the Campaign model on any entry in the group.
   - Return CampaignListResponse with all campaign summaries and total count.

3. Use `from negotiation.domain.types import NegotiationState` to check state values. Active states are: INITIAL_OFFER, AWAITING_REPLY, COUNTER_RECEIVED, COUNTER_SENT. Terminal states: AGREED, REJECTED. Special: ESCALATED, STALE.

4. Handle edge case: empty negotiation_states returns `{"campaigns": [], "total": 0}`.
  </action>

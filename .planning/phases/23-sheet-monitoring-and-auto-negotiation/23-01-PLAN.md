  <name>Task 1: Processed influencers SQLite table and SheetMonitor class</name>
  <files>

  <action>
1. In `src/negotiation/state/schema.py`, add a new function `init_processed_influencers_table(conn)` following the same pattern as `init_negotiation_state_table`. Table schema:
   ```sql
   CREATE TABLE IF NOT EXISTS processed_influencers (
       id INTEGER PRIMARY KEY AUTOINCREMENT,
       campaign_id TEXT NOT NULL,
       influencer_name TEXT NOT NULL,
       row_hash TEXT NOT NULL,
       processed_at TEXT NOT NULL DEFAULT (strftime('%Y-%m-%dT%H:%M:%SZ', 'now')),
       UNIQUE(campaign_id, influencer_name)
   )
   ```
   - `row_hash` is a SHA-256 hex digest of the serialized InfluencerRow data, used to detect modifications.
   - Add an index on `campaign_id` for efficient per-campaign lookups.

2. Create `src/negotiation/sheets/monitor.py` with class `SheetMonitor`:
   - `__init__(self, sheets_client: SheetsClient, conn: sqlite3.Connection)` -- stores references.
   - `_compute_row_hash(self, row: InfluencerRow) -> str` -- SHA-256 of `row.model_dump_json()`.
   - `_get_processed(self, campaign_id: str) -> dict[str, str]` -- returns `{influencer_name: row_hash}` from `processed_influencers` table for a campaign.
   - `_mark_processed(self, campaign_id: str, name: str, row_hash: str) -> None` -- INSERT OR REPLACE into `processed_influencers`.
   - `check_campaign_sheet(self, campaign: Campaign) -> SheetDiff` -- The core method:
     a. Call `self._sheets_client.get_all_influencers(worksheet_name=campaign.influencer_sheet_tab or "Sheet1", spreadsheet_key_override=campaign.influencer_sheet_id)`.
     b. Get processed dict via `_get_processed(campaign.campaign_id)`.
     c. For each InfluencerRow from the sheet:
        - Compute row_hash.
        - If name NOT in processed: it is a **new** row.
        - If name IS in processed but hash differs: it is a **modified** row.
        - If name IS in processed and hash matches: skip (no change).
     d. Return a `SheetDiff` dataclass with `new_rows: list[InfluencerRow]`, `modified_rows: list[tuple[InfluencerRow, str]]` (row + old_hash), `campaign: Campaign`.
   - `mark_rows_processed(self, campaign_id: str, rows: list[InfluencerRow]) -> None` -- public method to mark rows as processed after successful negotiation start. Calls `_mark_processed` for each row.

   Define `SheetDiff` as a simple dataclass at module level:
   ```python
   @dataclass
   class SheetDiff:
       campaign: Campaign
       new_rows: list[InfluencerRow]
       modified_rows: list[tuple[InfluencerRow, str]]  # (row, old_hash)
   ```

   Handle errors gracefully: if `get_all_influencers` raises `ValueError` (empty sheet) or `gspread` exceptions, log a warning and return an empty SheetDiff. Do NOT crash the polling loop.
  </action>
  <verify>
    `python -c "from negotiation.sheets.monitor import SheetMonitor, SheetDiff; print('import ok')"` succeeds.
    `python -c "from negotiation.state.schema import init_processed_influencers_table; print('ok')"` succeeds.
  </verify>
  <done>
    SheetMonitor class exists with check_campaign_sheet() that returns SheetDiff with new/modified rows.
    processed_influencers table DDL function exists in schema.py.
    Row hashing uses SHA-256 of model_dump_json().
    Already-processed rows with unchanged hash are excluded from results.
  </done>
